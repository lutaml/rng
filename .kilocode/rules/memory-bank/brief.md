The Rng Ruby library is a parser and generator for the RNG (Relax NG) and RNC
(Relax NG Compact Syntax) schema languages. It allows users to read as a DOM,
manipulate, and generate RNG/RNC schemas programmatically (through a Builder DSL
interface) within Ruby applications.

The library provides functionality for parsing RNG files and validating RNG/RNC
schemas.

It also supports converting RNG <=> RNC schemas, and converting to the XSD (XML
Schema Definition) format.

This library uses the Parslet parsing library for parsing RNG/RNC schemas.
It is part of the Lutaml suite of modeling tools.
